Wymagane środowisko Unity w wersji 2019.2.0f1

1. Zainstaluj odpowiednią wersję środowiska
2. Zainstaluj następnie w środowisku Unity dodatek PUN2 (Photon)
3. Aby korzystać z wersji aplikacji singleplayer uruchom grę w Unity (za pomocą przycisku play)
4. Aby korzystać z wersji multiplayer uruchom grę w osobnym okienku (CTRL+B) oraz w Unity tak, aby jednocześnie aplikacja miała dwie instancje.
